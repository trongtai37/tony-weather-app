# Tony Weather App
### A Simple Weather App to practice ReatJS

### Something I use to build this App:
  * HTML, CSS, JavaScript.
  * ReactJS, JSX, Webpack.
  * Yarn, Git.
  * Getting data from Current Weather Data's API

### How to get this App:
```
  git clone https://trongtai37.github.io/tony-weather-app.git
```
### How to use this App:
```
  git clone https://trongtai37.github.io/tony-weather-app.git
  cd tony-weather-app
  yarn 
  yarn run build
```
